# Examples

Copy-paste ready MCP client configs.

| File | Where it goes |
|------|---------------|
| `claude-code.json` | `~/.claude.json` (merge `mcpServers`) — or use `claude mcp add` CLI |
| `cursor-mcp.json` | `~/.cursor/mcp.json` (global) or `.cursor/mcp.json` (project) |
| `claude-desktop.json` | macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`<br>Windows: `%APPDATA%\Claude\claude_desktop_config.json` |
| `vscode-mcp.json` | `.vscode/mcp.json` in your project root |
| `zed-settings.json` | `~/.config/zed/settings.json` (merge `context_servers`) |
| `prompts.md` | Example prompts to try once installed |

For one-click install (VS Code, Cursor) see the badges in the [main README](../README.md).
